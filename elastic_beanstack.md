# Elastic BeanStack


> Platform as a Service

- Similar to Heroku, but more configurable
- supports NodeJS
- Server logs to **S3**
- Manages capacity, load-balancing, etc.
- Uses *git* to deploy
- uses micro **EC2** Amazon Linux **AMI**
- Auto scaling
- container based
- can be used for *smoke tests*
